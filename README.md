# Data Structure & Object-Oriented Programming using Java

## Description

The Data Structures and Object-Oriented Programming course covers concepts and efficient techniques for organizing, storing, accessing, and manipulating data in computer programs. Students will learn to design and implement data structures, including arrays, linked lists, stacks, queues, hashing, maps, trees, and graphs. Additionally, this course provides an understanding of the fundamental principles of object-oriented programming (OOP), including classes, objects, attributes, and methods, as well as core OOP concepts such as abstraction, encapsulation, inheritance, and polymorphism. Students will develop a strong grasp of these concepts through both theoretical study and hands-on programming practice in Java.

## Course Learning Objectives

The learning objectives associated with this course are:

1. Students are able to explain and implement various data structures.
2. Students are able to explain and implement object-oriented programming concepts.
3. Students are able to analyze data structure efficiency using Big-O notation to evaluate and select appropriate data structures.
4. Students are able to design and implement a program using an object-oriented programming paradigm and appropriate data structures to solve the given problem.
